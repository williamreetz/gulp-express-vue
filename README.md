# Gulp-Express-Vue
A starterkit for gulp, express and vue developer.

## Requirements
[Node.js](https://nodejs.org/en/)

## Installation

```sh
# clone repository
git clone git@github.com:williamreetz/gulp-express-vue.git

# change to project directory
cd gulp-express-vue

# install dependencies
npm install

# run application
gulp
```
